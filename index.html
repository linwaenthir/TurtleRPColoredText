How to replace your in-game name:

  • First, make sure you're not in-game and locate your TurtleWoW installation folder.
  • Once found, navigate to 'WTF/Account/[account_name]/[server_name]/[character_name]/SavedVariables/', replacing the bracketed sections with the relevant info.
  • Find the file called 'TurtleRP.LUA' and open it with any text-editor. (You may wish to back-up this file.)
  • With the current release (1.3.4), you'll see two sections:
    • 'TurtleRPCharacterInfo', which contains the data for your active profile. You'll need to edit this if you're changing the name on your active profile.
    • 'TurtleRPPlayerProfiles', which contains a numbered list of your four saved profiles. You'll need to edit the entry that corresponds with the profile you wish to change.
  • Find the line labeled '[full_name] = "Your Name Here";' in the profiles you need to edit.
  • Replace the information within the quotes (i.e, "Your Name Here") with the copied text code above. (Note: The code MUST be pasted within the quotes, with a semicolon ';' at the end.)
  • Save the file and load your game! (You may need to swap profiles to see the effect.)
